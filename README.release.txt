frangiclave-mod-manager
=======================

Requirements
------------

- Windows:
    You will need the .NET Framework, which should already be installed.
    You can download it from here: https://dotnet.microsoft.com/download
- macOS and Linux:
    You will need to install Mono.
    You can download it from here: https://www.mono-project.com/download/stable/

Running
-------

- Windows:
    Double-click FrangiclaveModManager.exe to run it.
- macOS and Linux:
    Open a terminal, navigate to the directory, and run:
    mono FrangiclaveModManager.exe

Setup
-----

On first run, you will need to specify the location of your game installation on
the command-line. You can find this location through Steam's and GOG's clients.
This location will then be saved for later runs.

Usage
-----

You will need to re-run the manager every time the game is updated in order to
patch it.
